But du jeu

Le but du jeu est de ramasser les fruits qui apparaissent sur le terrain en déplaçant votre serpent.

Comment jouer ?

Cliquer sur l'écran de la console pour lancer le jeu
Appuyer sur la touche espache pour lancer une partie
Utiliser les flèches directionnelles pour déplacer le serpent
